# Changelog — July 2026

**Mobile release:** 1.0.21 (44)  
**Focus:** Synced lyrics/captions, radio rights opt-in, Networx Home, Nearby People, Discover video creation, push/app updates

---

## Lyrics & captions

- **Artist lyrics upload:** Optional plain-text lyrics on web + mobile upload; backend force-aligns to audio (ElevenLabs Forced Alignment).
- **Auto-transcription fallback:** Approved tracks without lyrics can be transcribed (ElevenLabs Scribe); stored with `auto_generated` flag.
- **Players:** Synced lyrics panels on web and mobile; admin **Lyrics** studio; artist edit; listener Library read-only player with seek.
- **Env:** `ELEVENLABS_API_KEY` (Railway backend). Optional `ELEVENLABS_STT_MODEL_ID` (default `scribe_v1`).
- **Migrations:** `song_lyrics`, `song_lyrics_alignment`, `song_lyrics_auto_generated`, `get_artist_daily_likes`.

## Full-Song Radio Opt-In

- **Legal:** New page `/full-song-radio-opt-in` (Legal Center → Features & programs); Artist Agreement §5 links to the addendum.
- **Upload UI (web + mobile):** Required radio authorization checkbox; separate DJ livestream checkbox; optional archived DJ mixes checkbox.
- **DB / API:** `songs.opt_in_full_song_radio`, `opt_in_dj_livestreams`, `opt_in_dj_archived_mixes`; create requires radio opt-in.

## Mobile product surfaces

- **Networx Home** as post-sign-in landing; Radio tab remains the full player.
- **Gem dashboard** for artist tools / studio shortcuts.
- **Nearby People:** `users.city` / `zip_code`, geocoding, map pins, directory lists, GPS prompt.
- **Discover:** Reshuffle, artist search, liked/disliked library; **create-video** to liked clips (front selfie camera, countdown); feed storage limit **75MB**.
- **Nav:** Nearby People primary; Room renamed **The Chat Room**; notifications inbox + bell restored.
- **Push:** `user_device_tokens`, expanded alerts, first-login permission prompt; `app_releases` for update prompts.

## Backend / infra

- **Migrations applied (Jul):** device tokens + app releases, users city/zip, feed bucket 75MB, artist daily likes (plus earlier lyrics / radio opt-in).
- **Railway MCP:** Use Cursor `user-railway` tools (`whoami`, `list_projects`, `list_services`, `list_deployments`, `get_logs`, `deploy`, `set_variables`, …).
