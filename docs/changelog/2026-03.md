## Summary

March 2026 updates: streamer approval flow, Stream settings as a dedicated menu, Discover (Instagram-style tabs), web bottom nav, and Live discovery.

## Streamer approval and Stream settings

- **Streaming requires admin approval**: Artists and Catalysts no longer get go-live automatically. They must **request streaming access**; an admin approves in **Admin → Streamers**.
- **Backend**:
  - Migration `022_streamer_approval.sql`: `artist_live_profiles` gains `streaming_applied_at`, `streaming_approved_at`, `streaming_rejected_at`.
  - `GET /artist-live/streamer-status`: current user’s canStream, appliedAt, approvedAt, rejectedAt.
  - `POST /artist-live/apply`: submit application (artist or Catalyst only).
  - `startLive()`: only allowed if `streaming_approved_at` is set (or user is admin).
  - Admin: `GET /admin/streamer-applications`, `PATCH /admin/streamer-applications/:userId` (approve/reject).
- **Web**: **Stream settings** at `/stream-settings` and in sidebar (More) for artists/Catalysts. Request access, pending/rejected states, and (when approved) Stream Manager + Live services link. Profile Go Live card links to Stream settings.
- **Mobile**: **Stream settings** in More sheet; `StreamSettingsScreen` for request/pending/rejected/approved (then Open Stream Manager). Profile Go Live card opens Stream settings. `LivestreamService`: `getStreamerStatus()`, `applyToStream()`.

## Discover (Instagram-style) and profiles

- **Discover** (`/discover`): Tabs **For you** (all), **Artists**, **Catalysts** (line variant, primary accent). Same discovery API and `/artist/[id]` profile route for both. Search and filters (location, service type, nearby/price for Catalysts). Card hover and primary-tinted buttons for branding.

## Web UI: bottom nav and Live

- **Bottom nav**: Fixed bottom bar (Twitch/IG-style): Home (Dashboard), Discover, Live, Activity (Messages), Profile. Active item uses primary color; content has bottom padding so it isn’t hidden.
- **Live page** (`/live`): Lists live sessions from `GET /artist-live/sessions`; sort (Recommended, Viewers high/low, Recently started); cards link to `/watch/:artistId`.
- **Stream Manager** (GoLiveSheet): Edit stream info (title, description, category), Start/End stream; used from profile (when approved) and from Stream settings.
