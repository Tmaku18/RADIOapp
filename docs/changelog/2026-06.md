# Changelog — June 2026

**Mobile release:** 1.0.12 (28)  
**Focus:** Hard radio sync, Pro-Networx subscription gates, mobile player reliability, visualizer parity

---

## Radio sync and playback

- **Hard live sync (web + mobile):** Removed mid-song “DVR hold” that let devices stay on different songs. Playing listeners always follow the server’s current track; explicit pause still respected.
- **Background playback fixes:** Web radio no longer repeats or stalls when tab is backgrounded; mobile background track-change sync improved.
- **Crossfade removed:** Legacy crossfade archived; direct track transitions for true-radio feel.
- **Skip controls removed:** Core live radio has no user skip (web + mobile).
- **DJ Mixes station:** Added `us-dj-mixes` across web, backend, and mobile.

## Pro-Networx (networking & monetization)

- **Resume access:** Private `resumes` bucket paths signed before public profile return (fixes 404).
- **Subscription gate:** Only owner, admins, and active Pro-Networx subscribers receive signed resume URLs (contact info protection).
- **Resume UX:** Resume button always visible; non-subscribers routed to subscribe (web checkout, app paywall sheet).
- **Messaging UX:** Same pattern as resume — Message button always visible; `messagingLocked` routes to subscribe flow.
- **Backend flags:** `resumeLocked` and `messagingLocked` on public profiles.

## Mobile player and Dimension

- **3D hero on devices:** WebView loads web `/embed/butterfly` for parity on physical Android (abandoned native `flutter_angle` on device).
- **Album covers:** 2D `Image.network` replaces `three_js` cube (GL init failure left covers invisible on devices).
- **Visualizer:** Organic simulated motion on mobile; web FFT dramatization with bass reactivity fixes.
- **Builds shipped:** 1.0.9 (25) → 1.0.10 (26) sync → 1.0.11 (27) album art → 1.0.12 (28) Pro-Networx gates.

## Web

- Player click-through and music-like visualizer on mobile web.
- Clip and sample previews mutually exclusive.
- Radio track replay at end suppressed when tab hidden.

## Analytics (carried from late May)

- Listens vs Ears Reached unified across web, mobile, marketing.
- Pro-Networx marketing stats public API + SSR fixes.
