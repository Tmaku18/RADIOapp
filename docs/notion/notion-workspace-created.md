## How to add the radio icon (ðŸ“») in Notion

1. Open the page or database (use any link below, or open it from the sidebar).
2. At the top, next to the page title, click the **current icon** (or the empty icon area if there is no icon).
3. In the icon picker, either:
   - Paste or type the emoji **ðŸ“»**, or
   - Search for **"radio"** and select the radio emoji.
4. Repeat for each page and database you want to brand (e.g. start with the 5 top-level pages, then the databases and templates).
